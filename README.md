**Note:** This is an early project from my web development journey, created to practice HTML, SCSS, JavaScript, and Webpack fundamentals.

A responsive, multi-page corporate website for **Sky Company** - a web development company based in Lebanon. The website showcases company services, team members, projects portfolio, blog posts, and contact information with a modern, clean design.

---

## Features

- **Responsive Navigation** - Bootstrap navbar with active state highlighting across all pages
- **Hero Carousel** - Full-width image slider with overlay text and call-to-action buttons on homepage
- **Projects Showcase** - Interactive carousel displaying latest company projects with hover effects
- **Blog System** - Blog listing with search functionality and detailed post pages
- **Image Gallery** - Modal-based image gallery for project details (Project Details page)
- **Customer Testimonials** - Carousel showcasing client feedback
- **Fully Responsive** - Mobile-first design using Bootstrap 4 grid system
